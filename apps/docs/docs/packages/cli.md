# @azmr/cli

Command-line tooling for Azmara Platform.

## Usage

```bash
npx azmara <command>
```

## Commands

| Command | Status | Description |
|---|---|---|
| `azmara version` | ✅ Available | Print CLI version |
| `azmara help` | ✅ Available | Show help message |
| `azmara init` | 🚧 Coming soon | Scaffold a new Azmara app |
| `azmara fix <file>` | 🚧 Coming soon | Run AI auto-fix on a file |
| `azmara db:query` | 🚧 Coming soon | Query a local SQLite database |

## Examples

```bash
# Check version
azmara version

# Get help
azmara help
```

## Roadmap

The CLI is the entry point for the `create-azmara-app` scaffolding experience — planned for the SaaS layer phase.
